## Specification

| Property | Guideline |
|:---|:---|
| **Format** | PNG for screenshots and diagrams, JPG acceptable for photographs |
| **Hero banner** | 1600 × 500 px approx., readable when scaled to full README width |
| **Gallery images** | 1200 px wide minimum, consistent aspect ratio within a row |
| **App screenshots** | Device frame optional but consistent across all screens |
| **File size** | Under 500 KB each — compress before committing |
| **Background** | Neutral; avoid pure white, which glares against GitHub dark mode |

> [!WARNING]
> Before committing any photograph, verify it does not expose proprietary detail: visible PCB traces, component markings, terminal wiring, labels with credentials or network names, or readable schematics on a screen in frame. Crop or blur anything that does.

> [!NOTE]
> Photographs of the site should not reveal precise location detail if the aqueduct administration considers that sensitive.
